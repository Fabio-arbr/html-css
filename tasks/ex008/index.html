Principais Formatações

Em Negrito

Para deixarmos uma texto em Negrito basta usarmos a tag B (não semântico)

Para deixarmos um texto em destaque basta usarmos a tag STRONG (semântico)

Itálico

Temos nesta frase um tertom em itálico para isto basta usarmos a tag I(não semântica)

Nesta frase temos um termo em ênfase utilizando a tag EM(semântica)

Marcaçãp de texto

É possível criar também um texto marcado utilizando a tag MARK

Texto grande e pequeno

Aqui está ambas as representações. O texto grande é a tag BIG, ela se tornou obseleta.
Para criarmos um texto pequeno basta utilizarmos a tag SMALL

Texto deletado

Podemos marcar um texto como excluído (indicando assim que ele deve ser lido, mas não considerado) utilizando a tag DEL.

texto inserido

Podemos também marcar um texto como inserido (para dar uma ênfase e indicar q ele foi adicionado), para adicona-lo basta utilizarmos a tag INS.(semântica)

existe também o sublinhado utilizando a tag U ,no entanto ela é obsoleta.(não semântica)
